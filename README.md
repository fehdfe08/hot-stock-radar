# Hot Stock Radar

东方财富股吧人气榜 Top10 热门股统计与分析看板。

## 功能

- 抓取 A 股人气榜 Top10
- 补充行情、行业、概念、粉丝结构
- 筛选实质催化：业绩、订单/中标、项目投产、回购增持、政策与研报
- 剔除低价值行情记录：龙虎榜、融资融券、盘口异动、简单涨跌播报
- GitHub Actions 每小时生成最新数据并发布到 GitHub Pages

## 本地运行

```bash
npm run build:data
npm run serve
```

然后打开 `http://localhost:4173`。
